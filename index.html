HTML Tags

This table showcases HTML tags and their respective meanings.

HTML Tags and their Descriptions
Tag Description
<!DOCTYPE html> Defines the document type, ensuring proper rendering by browsers.
<html> Root element of an HTML document. Contains the entire document structure.
<head> Contains meta-information about the document, like title, stylesheets, and scripts.
<body> Contains the visible content of the document. Contains all the HTML elements, like headings, paragraphs, images, and links
<h1> to <h6> Heading elements with increasing font sizes (h1 to h6)
<p> Paragraph element, used to define a paragraph of text.
<br> Line break element, used to create a line break in a paragraph.
<a> Hyperlink element, used to create links to other web pages or files.
<img> Image element, used to embed images in documents.
<ul> and <ol> Unordered and ordered lists, respectively.
<li> List item element, used to define list items within lists.
<table>, <tr>, <th>, <td> Table element, table row element, table header cell element, and table data cell element, respectively.
<form>, <input>, <textarea>, <select>, <option> Form element, input element, textarea element, select element, and option element, respectively.
<div> and <span> Div element, span element, used to create block-level or inline elements, respectively.
<header>, <footer>, <nav> Header element, footer element, and navigation element, respectively.
<section>, <article>, <aside> Section element, article element, and aside element, respectively.
<main> Main content element, containing the main content of a document.
<time> Time element, used to represent a specific time or date.
<mark> Mark element, used to highlight text.
<code> Code element, used to display computer code.
<pre> Preformatted text element, used to display text in a fixed-width font.
<blockquote> Blockquote element, used to create a section quote.
<hr> Horizontal rule element, used to create a thematic break in a document.
<details>, <summary> Details element, summary element, used to create a collapsible section with additional details.
<figcaption> Figcaption element, used to provide a caption for a figure.
<figure>, <figcaption> Figure element, figcaption element, used to group related elements like images, videos, and figures.
<svg>, <path>, <circle>, <rect> SVG element, path element, circle element, and rectangle element, respectively.
<audio>, <video> Audio element, video element, used to embed audio and video content.
<canvas> Canvas element, used to create graphics on a web page.
<script> Script element, used to embed executable JavaScript code.
<noscript> No script element, used to display content to users who disable scripts.
<template> Template element, used to define a content section that will be cloned when the template element is inserted into the DOM.
<slot> Slot element, used to define a placeholder for content that can be dynamically filled in by a parent element.
<slot-name> Slot name element, used to define a named slot in a parent element that can be dynamically filled in by a child element.
<style> Style element, used to define style information for HTML elements.
<link> Link element, used to link external resources like stylesheets and scripts.
